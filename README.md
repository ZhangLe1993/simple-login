# 美观好看的登录模板 simple-login 
    可以下载下来集成到自己的项目中哦, 只需准备一个登录按钮, 一个注册按钮, 其他都是我在AccessPanel组件中封装好的.
    项目环境: vue
    弹框依赖 element-ui  
    cnpm i element-ui -S
